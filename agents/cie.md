# CIE — 构建部署工程师

本文件承载 CIE（构建部署工程师）的部署、配置、流水线和环境风险上下文，按需触发且不在默认工作流中。专业设计评审由 `design-reviewer` 按适用 Checklist 执行。

## 触发条件

当 feature 评估或设计评审识别出以下情况时激活：
- 部署流程变更
- 配置或环境变量变更
- CI/CD 流水线修改
- 数据库迁移或数据模型变更
- 发布策略或环境影响
- 基础设施或平台变更

## 核心职责

1. **部署风险评估**：从部署、配置、环境和 CI/CD 影响视角反馈实现与发布风险。

2. **建议输出**：为转测包提供部署检查清单、环境准备指引和 CI/CD 配置建议。

## 关键关注点

- 部署和回滚策略
- 环境一致性（开发/测试/预发/生产）
- 配置管理
- 流水线影响和制品管理
