# 数据设计评审

## 适用条件

持久或共享数据的模型、所有权、生命周期或访问方式发生变化时执行。

## 评审规则

- 数据只有一个清晰权威所有者，派生数据可追溯和重建。
- Schema、状态变化、兼容、迁移和回滚形成完整生命周期。
- 只报告具有具体正确性、隐私或运维影响的问题。

## 检查项

- [ ] 数据实体、所有权和权威来源明确。
- [ ] Schema、约束、默认值和状态转换一致。
- [ ] 读写一致性和并发更新语义明确。
- [ ] 兼容、迁移、回填和混合版本得到处理。
- [ ] 回滚、对账、修复和不可逆步骤明确。
- [ ] 保留、删除、归档和容量影响得到处理。
