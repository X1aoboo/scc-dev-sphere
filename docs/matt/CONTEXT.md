# Feature Design 领域语言

本文定义插件在协作式 Feature Design 中使用的统一术语。

## 术语

**设计段落（Design Section）**：
为增量确认而向用户呈现的一组内聚设计内容；其篇幅随问题复杂度缩放。
_避免使用_：设计节点、流程节点

**已确认设计（Confirmed Design）**：
用户在当前协作中已经接受的设计内容。修改时必须明确说明并重新确认，但不表示系统需要保存依赖图或保证会话无损恢复。
_避免使用_：冻结状态、锁定节点

**基线产物（Baseline Artifact）**：
一个设计阶段正式批准后发布、可供下游阶段依赖的设计结果。
_避免使用_：Work、Draft、设计状态
